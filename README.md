# ♻️ 우리동네 클린-링크 (Clean-Link)

공공데이터를 활용한 종량제 봉투 판매처 검색 및
지역 맞춤형 쓰레기 배출 가이드 서비스

## 프로젝트 목적

- 종량제 봉투 판매처 조회
- 지역별 배출 규정 안내
- 판매점 밀집도 분석
- 환경 사각지대 시각화

## 사용 기술

Frontend
- React Native

Backend
- FastAPI

Database
- PostgreSQL + PostGIS

API
- 행정안전부 종량제 봉투 판매업 조회 API

## 기대효과

- 과태료 예방
- 정보 접근성 향상
- 환경 정책 데이터 제공
