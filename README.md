# ♻️ 우리동네 클린-링크 (EcoData Hub)

공공 데이터를 활용하여 지역별 환경 정보와 재활용/판매점 현황을 시각화한 플랫폼입니다.

## 🛠 사용 기술
- HTML5, CSS3, JavaScript
- Chart.js (데이터 시각화)
- Web Geolocation API (위치 기반 서비스)

## 📊 주요 기능
- **현재 위치 확인:** 사용자 위치 기반 위도/경도 확인
- **지역별 데이터 분석:** 강남, 서초 등 주요 지역의 판매점 수를 그래프로 시각화
- **환경 통계 대시보드:** 지역별 밀집도와 평균 가격 정보 제공
